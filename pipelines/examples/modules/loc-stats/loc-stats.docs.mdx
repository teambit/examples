---
description: A build task that generates LOC stats for a component
labels: ['demo']
---

Example:

```json
{
  "total": 0,
  "source": 0,
  "comment": 0,
  "single": 0,
  "block": 0,
  "mixed": 0,
  "empty": 6,
  "todo": 0,
  "blockEmpty": 0
}
```

Use `bit artifacts --task LocStats --out-dir loc-stats-output` to extract the output into your workspace.
