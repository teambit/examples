'html-s3' aspect bundles plain JS Components and deploys them to s3 hosting.

## Usage

Add an app type setup file to your Component. For example:

```bash
$ cd path/to/my-component
$ touch my-component.html-s3.ts
```

Add your app's info. For example:

```ts
import { HtmlS3Options } from '@teambit/apps.examples.aspects.html-s3';

const s3Config = {
  credentialsProfile: 'default',
  bucket: 'my-bucket',
  region: 'us-east-1',
};

const plainJS: HtmlS3Options = {
  name: 'my-component-app',
  entry: [require.resolve('./my-component.app-root')],
  s3Config,
};

export default plainJS;
```

## AWS config

### Create a bucket and credentials

1. Create an s3 bucket
2. [Create a shared credentials file](https://docs.aws.amazon.com/sdk-for-javascript/v3/developer-guide/loading-node-credentials-shared.html)

### Set you bucket for static site hosting

1. [Set public read access](https://docs.aws.amazon.com/AmazonS3/latest/userguide/WebsiteAccessPermissionsReqd.html)
2. Set you bucket as static website hosting:
   1. Go to your bucket (https://s3.console.aws.amazon.com/s3/buckets/YOUR_BUCKET)
   2. Go to 'Properties'
   3. Go to 'Static website hosting'
   4. Choose 'edit' > 'enable'
   5. Set `index.html` as the index document
